Consultar Peças:

1: O utilizador seleciona a opção "Peças" na janela "AutoSell".

2: O sistema mostra a janela "Gerir Peças" com os dados das peças.

3: O utilizador seleciona uma peça na lista de peças e clica no botão "Consultar" no ecrã "Gerir Peças".

4: O sistema mostra a "Janela Consultar Peça" com os dados da peça selecionada.

5: O utilizador verifica a Peça selecionada.

Caminhos alternativos: 

3.1: O utilizador clica no botão "Voltar"
3.1.1: O sistema fecha a janela "Gerir Peças".

5.1: O utilizador clica no botão "Voltar"
5.1.1: O sistema fecha a janela "Gerir Peças".




Adicionar Peças:

1: O utilizador seleciona a opção "Peças" na janela "AutoSell"

2: O sistema mostra a janela "Gerir Peças"

3: O utilizador seleciona o butão "adicionar" na janela "Gerir Peças"

4: O sistema mostra a janela "Adicinar Peças"

5: O utilizador preenche os campos com os dados da peça e carrega no botão "salvar".

6: O sistema valida os dados inseridos.

7: O sistema guarda os dados e fecha a janela "Adicionar Peças"

Caminhos alternativos:

3.1: O utilizador clica no botão "Voltar"
3.1.1: O sistema fecha a janela "Adicinar Peças".

5.1: O utilizador não preenche os campos obrigatórios e clica no botão "Salvar".
5.2.1: O sistema mostra uma mensagem de erro e volta á janela "Gerir Peças".




Editar Peça:

1: O utilizador seleciona a opção "Peças" na janela "AutoSell"

2: O sistema mostra a janela "Gerir Peças"

3: O utilizador seleciona uma peça na lista de peças e clica no botão "Editar" no ecrã "Gerir Peças".

4: O utilizador preenche os campos com os dados da peça e carrega no botão "salvar".

5: O sistema recebe os dados da janela, guarda-os, atualiza a lista de eventos no ecrã "Gerir Eventos"
, mostra uma mensagem de sucesso e fecha a janela "Editar Peça".

Caminhos alternativos: 

4.1: O utilizador clica no botão "Voltar"
4.1.1: O sistema fecha a janela "Adicinar Peças".

5.1: O utilizador não preenche os campos obrigatórios e clica no botão "Salvar".
5.2.1: O sistema mostra uma mensagem de erro e volta á janela "Gerir Peças".




Remover Peças:

1: O utilizador seleciona a opção "Peças" na janela "AutoSell"

2: O sistema mostra a janela "Gerir Peças"

3: O utilizador seleciona uma peça na lista de peças e clica no botão "Apagar" no ecrã "Gerir Peças".

4: O sistema elimina a peça selecionada, remove-a da lista de peças no ecrã "Gerir Peças" 
e mostra uma mensagem de sucesso

Caminhos alternativos: 

3.1: O utilizador clica no botão "Apagar" sem selecionar uma peça na lista de peças.
3.1.1: O sistema mostra uma mensagem de erro e o volta á janela "Gerir Peças".




Realizar Pedido De Peças:

1: O utilizador seleciona a opção "Peças" na janela "AutoSell"

2: O sistema mostra a janela "Gerir Peças"

3: O utilizador clica no butão "lista de Pedidos"

4: O sistema mostra a janela "Lista de pedidos"

5: o utilizador clica no butão "pedir"

6: O sistema abre a janela "Pedir Peças"

7: O sistema preenche a lista de peças com todas as peças incluidos na lista passada
 a esta janela.

8: O utilizador seleciona a loja requerente e a loja fornecedora.

9: O utilizador clica no butão "seta direita" movendo a peça selecionada da lista de peças
 para a lista de peças do pedido.

10: O sistema mostra a janela "Quantidade peça".

11: o ultizador preenche o campo de quantidade.

12: O utilizador clica no botão "Pedir" na janela "Pedir Peças".

13: O sistema guarda a lista de peças selecionados e fecha a janela "Pedir Peças".


Caminhos alternativos:

3.1: O utilizador clica no botão voltar.
3.2.1: O sistema fecha esta janela volta á janela "Gerir Peças".

8.1:  O utilizador seleciona lojas iguais na "loja requerente" e "loja fornecedora"
8.1.1: O sistema mostra uma mensagem de erro

10.1:  O utilizador não preenche o campo quantidade na janela "Quantidade peça"
10.1.1: O sistema mostra uma mensagem de erro

12: O utilizador não selecionou nenhuma peça para a "lista de peças do pedido".
12.1.1: O sistema mostra uma mensagem de erro.

4: O sistema mostra a janela "Lista de pedidos"

4: O utilizador clica no botão "Consultar" no ecrã "Gerir Clientes".

5: O sistema mostra a janela "Consultar Cliente".

Caminhos alternativos:
